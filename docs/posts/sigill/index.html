记录Chrome 网页崩溃

环境

  • Ubuntu 22.04 LTS GNOME
  • Chrome
  • 16 Core vCPU
  • 32G Memory

在做浏览器CPU挖矿的时候，Chrome网页经常会隔几分钟就崩溃，报SIGILL错误 在搜索一番后，猜测可能是由于内存限制

解决方法 Terminal 启动chrome，并添加参数

google-chrome --js-flags="--max_old_space_size=8192"  #size 为分配的内存大小
